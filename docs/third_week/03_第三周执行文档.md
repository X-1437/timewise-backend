### 阶段3：DAL 层 - 抽象接口（约0.5小时）

#### 任务3.1：定义抽象基类
创建 `dal/base.py`：
```python
from abc import ABC, abstractmethod
from typing import List, Optional
from models.session import Session
from models.message import Message
from models.file import File

class BaseSessionDAL(ABC):
    @abstractmethod
    async def create(self, session: Session) -&gt; Session:
        pass
    
    @abstractmethod
    async def get_by_id(self, session_id: str) -&gt; Optional[Session]:
        pass
    
    @abstractmethod
    async def get_all(self) -&gt; List[Session]:
        pass
    
    @abstractmethod
    async def update(self, session: Session) -&gt; Session:
        pass
    
    @abstractmethod
    async def delete(self, session_id: str) -&gt; bool:
        pass

# 同理定义 BaseMessageDAL 和 BaseFileDAL
```

**产品经理要理解**：这一层定义了"接口"，未来换数据库只需写新的实现，上层代码不用改！

### 阶段4：MongoDB 环境和连接（约1小时）

#### 任务4.1：安装和启动 MongoDB
**Windows：**
1. 下载 MongoDB Community Server：https://www.mongodb.com/try/download/community
2. 安装时选择"完整安装"
3. 安装后，MongoDB 服务会自动启动
4. 验证：打开浏览器访问 `http://localhost:27017`，应该看到 "It looks like you are trying to access MongoDB over HTTP"

**备选方案（如果安装困难）：**
使用 MongoDB Atlas 云数据库（免费版）
