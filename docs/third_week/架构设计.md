## 📖 完整项目例子：个人笔记应用

我们用同样的**个人笔记应用**来完整演示架构设计！

### 功能
- 创建笔记
- 查看笔记列表
- 查看单条笔记
- 删除笔记

## 第一步：确定架构模式

最简单的三层架构！

```
API层（Controller） → 业务逻辑层（Service） → 数据访问层（DAL） → 数据库
```

## 第三步：写代码（从头实现一遍）

### 3.1 数据模型（models/note.py）

定义数据结构：

```python
from pydantic import BaseModel
from datetime import datetime
from typing import Optional

class Note(BaseModel):
    id: Optional[str] = None  # MongoDB会自动生成_id
    title: str                # 标题
    content: str               # 内容
    created_at: Optional[datetime] = None
    updated_at: Optional[datetime] = None
```

---

### 3.2 数据访问层（dal/mongodb/note_dal.py）

封装数据库操作：

```python
from motor.motor_asyncio import AsyncIOMotorDatabase
from models.note import Note

class NoteDAL:
    def __init__(self, db: AsyncIOMotorDatabase):
        self.collection = db.notes

    async def create(self, note: Note) -> Note:
        # 插入数据库
        result = await self.collection.insert_one(note.model_dump())
        note.id = str(result.inserted_id)
        return note

    async def get_by_id(self, note_id: str) -> Note:
        # 根据ID查询
        doc = await self.collection.find_one({"_id": note_id})
        return Note(**doc) if doc else None

    async def get_all(self) -> list[Note]:
        # 查询所有，按创建时间倒序
        cursor = self.collection.find().sort("createdAt", -1)
        notes = []
        async for doc in cursor:
            notes.append(Note(**doc))
        return notes

    async def delete(self, note_id: str) -> bool:
        # 删除
        result = await self.collection.delete_one({"_id": note_id})
        return result.deleted_count > 0
```

---

### 3.3 业务逻辑层（services/note_service.py）

处理业务逻辑：

```python
from dal.mongodb.note_dal import NoteDAL
from models.note import Note

class NoteService:
    def __init__(self, note_dal: NoteDAL):
        self.note_dal = note_dal

    async def create_note(self, title: str, content: str) -> Note:
        # 创建笔记
        note = Note(
            title=title,
            content=content,
            created_at=datetime.now(),
            updated_at=datetime.now()
        )
        return await self.note_dal.create(note)

    async def get_note(self, note_id: str) -> Note:
        # 获取单条笔记
        return await self.note_dal.get_by_id(note_id)

    async def list_notes(self) -> list[Note]:
        # 获取笔记列表
        return await self.note_dal.get_all()

    async def delete_note(self, note_id: str) -> bool:
        # 删除笔记
        return await self.note_dal.delete(note_id)
```

---

### 3.4 API层（api/notes.py）

接收请求，返回响应：

```python
from fastapi import APIRouter
from services.note_service import NoteService

router = APIRouter()

@router.post("/notes")
async def create_note(title: str, content: str):
    note = await NoteService.create_note(title, content)
    return {"success": True, "data": note}

@router.get("/notes")
async def list_notes():
    notes = await NoteService.list_notes()
    return {"success": True, "data": notes}

@router.get("/notes/{note_id}")
async def get_note(note_id: str):
    note = await NoteService.get_note(note_id)
    return {"success": True, "data": note}

@router.delete("/notes/{note_id}")
async def delete_note(note_id: str):
    success = await NoteService.delete_note(note_id)
    return {"success": success}
```

### 3.5 入口文件（main.py）

组装起来：

```python
from fastapi import FastAPI
from motor.motor_asyncio import AsyncIOMotorClient
from api.notes import router as notes_router

app = FastAPI()

# 连接数据库
client = AsyncIOMotorClient("mongodb://localhost:27017")
db = client.note_app

# 注册路由
app.include_router(notes_router)

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

## 第五步：总结架构设计要点

1. **分层清晰**：API → Service → DAL → DB
2. **职责单一**：每层只干一件事
3. **依赖方向**：上层依赖下层
4. **易于测试**：可以单独测试每层
