### 3.1 MCP工具封装（mcp_tools.py）
实现了4个核心时间序列分析工具：
- **eda_analysis**：探索性数据分析
- **preprocessing**：数据预处理
- **feature_analysis**：特征分析
- **forecast**：时间序列预测

**关键代码示例：**

```python
# mcp_tools.py - 定义工具
@server.list_tools()
async def handle_list_tools():
    return [
        types.Tool(
            name="eda_analysis",
            description="执行探索性数据分析（EDA），包括数据概览、缺失值统计、基本统计量",
            inputSchema={
                "type": "object",
                "properties": {
                    "data_name": {"type": "string", "description": "数据集名称"}
                },
                "required": ["data_name"]
            }
        ),
        # ... 更多工具
    ]

# mcp_tools.py - 执行工具
@server.call_tool()
async def handle_call_tool(name, arguments):
    if name == "eda_analysis":
        data_name = arguments.get("data_name", "未知数据")
        result = f"【探索性数据分析结果】数据集：{data_name}..."
        return [types.TextContent(type="text", text=result)]
```

### 3.2 大模型接入（llm_client.py）
- 支持OpenAI API接入
- 支持规则匹配（无API Key时也能工作）
- 意图识别功能
- 使用json.loads()替代eval()，修复安全问题

**关键代码示例：**

```python
# llm_client.py - 两种模式切换
def analyze_intent(self, query):
    # 模式A：真实OpenAI API（需要API Key）
    if os.getenv("OPENAI_API_KEY") and os.getenv("OPENAI_API_KEY") != "demo-key":
        try:
            response = self.client.chat.completions.create(
                model=self.model,
                messages=[...],
                tools=tools,
                tool_choice="auto"
            )
            # ... 处理返回结果
        except Exception as e:
            print(f"OpenAI API调用失败，使用规则匹配：{e}")
    
    # 模式B：规则匹配（默认，无需API Key）
    return self._rule_based_intent(query)

# llm_client.py - 规则匹配实现
def _rule_based_intent(self, query):
    query_lower = query.lower()
    if any(keyword in query_lower for keyword in ["预测", "forecast"]):
        return {"tool": "forecast", "args": {...}}
    elif any(keyword in query_lower for keyword in ["特征", "feature"]):
        return {"tool": "feature_analysis", "args": {...}}
    else:
        return {"tool": "eda_analysis", "args": {...}}
```

**重要说明**：
本次验证使用的是**规则匹配模式**（无需OpenAI API Key），核心逻辑验证完成。
如需接入真实OpenAI API，可在`.env`文件中配置`OPENAI_API_KEY`。
