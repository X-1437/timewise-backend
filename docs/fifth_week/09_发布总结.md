## 经验沉淀

1. **"先文档后代码"的流程有效**：所有功能改动都在用户确认 SRS 和执行手册后进行，减少了返工。
2. **结论优先的产品理念正确**：用户反复强调"不要让我自己总结"，倒推所有输出从技术报告转向产品级分析报告。
3. **统一规则优于特例补丁**：非分析输入从"补 3 个 case"到"白名单门禁"是质的提升。
4. **ZIP 离线包是正确选择**：纯 md 下载后图片不可见的问题被反复提出，ZIP 方案一次性解决。
