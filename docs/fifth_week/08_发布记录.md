## 发布内容

### 新增功能

| 编号 | 功能 | 描述 |
|---|---|---|
| FEAT-01 | 图片生成与展示 | 后端生成 PNG，前端 Markdown 渲染，支持放大预览 |
| FEAT-02 | 导出 ZIP 离线包 | 默认下载 ZIP（md + 图片 + 预处理 CSV），解压后本地可看图 |
| FEAT-03 | 一键全流程 | 输入"帮我运行全流程"自动执行 EDA→预处理→特征→三种预测→导出 |
| FEAT-04 | 结论优先输出 | 所有步骤（EDA/预处理/特征/预测）采用"结论→输出→明细"结构 |
| FEAT-05 | 非分析输入统一引导 | 白名单门禁，所有非分析输入统一返回引导回复 |
| FEAT-06 | 报告增强 | 时间间隔判断、数据样例、全中文化、重点加粗、看图结论 |

### 缺陷修复

| 编号 | 缺陷 | 修复内容 |
|---|---|---|
| FIX-01 | 中文字体方框 | matplotlib 添加中文字体回退 |
| FIX-02 | EDA 横轴按天显示 | 改为 MonthLocator + DateFormatter |
| FIX-03 | 图片点击下载非放大 | 前端 lightbox 放大预览 |
| FIX-04 | 预处理图不可读 | 添加噪音/异常点标注、摘要框、结论 |
| FIX-05 | 图例与摘要框重叠 | 图例移至右上角 |
| FIX-06 | 朴素预测方法映射错误 | 三种方法独立路由 |
| FIX-07 | 报告含无用字段 | 移除内部标识/UTC时间等 |
