# 第五周性能实测结果留痕

## 文档信息

- 文档版本：v1.0
- 编制时间：2026-06-25
- 测试环境：Windows / Python 3.x / FastAPI + Uvicorn / MongoDB localhost
- 测试数据：sample_timeseries.csv（37 行 × 4 列）
- 测试脚本：`scripts/smoke_performance.py`

---

## 实测结果

| 指标 | 实测值 | 阈值 | 结论 |
|---|---|---|---|
| 首次响应（EDA） | **290ms** | ≤ 800ms | ✅ 通过 |
| 端到端全流程 | **6.2s** | ≤ 15s | ✅ 通过 |
| 图表渲染（含 EDA 全步） | **≤333ms** | ≤ 2s | ✅ 通过 |

---

## 说明

- "首次响应"测量的是从发出"帮我做一下 EDA"请求到收到完整 EDA 回复（含图片生成）的时间。
- "端到端全流程"测量的是从"帮我运行全流程"请求到完成 EDA → 预处理 → 特征提取 → 三种朴素预测 → 导出报告的全部时间。
- "图表渲染"取 EDA 步骤内最重的一次调用（含图片生成）作为上限估计，实际纯图表渲染时间更短。
- 本地环境 MongoDB 和 MCP 子进程均在同一台机器上运行，生产环境需单独评估网络延迟。

---

## 结论

三项核心性能指标全部满足 SRS 验收标准，可作为发布评审与技术准入的支撑证据。
